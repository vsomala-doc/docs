Generating PDF Booklets

You can generate PDF files of all CRFs associated with specific participants or groups of participants enrolled in the selected study. Access PDF Booklets from the left menu on the EDC dashboard.

Use the available filters and customization options to tailor the content of the booklets. You can also track the status of previously generated booklets or download completed files directly from the ‚ÄòMy Previously Generated Booklets‚Äô section.

The following table details each filter available on the PDF Booklets page, including its description and purpose.

FilterDescription
SiteSelect one or more sites.
ParticipantsSelect one, multiple, or all participants.
CRFsSelect one, multiple, or all CRFs.

Exclude Non-Participant CRFsExclude CRFs with the ‚ÄòNon-Participant‚Äô switch enabled in Study Builder.

Exclude ‚ÄòTo Do‚Äô Status CRFsExclude CRFs in ‚Äòto do‚Äô status from the booklet, even if selected.

Generating PDF booklets‚Äã

The following steps outline how to generate PDF booklets.

Steps to generate PDF booklets:‚Äã
  1. From the EDC dashboard, select PDF Booklets. The PDF Booklets page appears.

  1. Select the site, participants, and CRFs from the drop-down menus.

  2. Check the relevant customization options to tailor the booklet.

  3. Select GENERATE. The generated PDF is added as a new row in the ‚ÄòMy Previously Generated Booklets‚Äô table.

Managing previously generated booklets‚Äã

The ‚ÄòMy Previously Generated Booklets‚Äô section allows you to track the status of your requests and download completed files.

Steps to manage previously generated booklets:‚Äã

  1. From the ‚ÄòPDF Booklets‚Äô page, scroll to the ‚ÄòMy Previously Generated Booklets‚Äô section.

  2. To check the status of a request, select REFRESH to update the table and check the status.

    NOTE: Requests are marked as ‚ÄòProcessing‚Äô while the booklet is being generated.

  3. Once a request is complete:

    • The status updates to ‚ÄòComplete‚Äô.
    • A download icon appears in the ‚ÄòAction‚Äô column.

  4. Select the download icon to download the generated booklet.
