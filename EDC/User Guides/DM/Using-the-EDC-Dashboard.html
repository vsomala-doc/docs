Using the EDC Dashboard

The EDC dashboard is your primary workspace for tracking tasks and accessing study data. It opens with the ‘Data Cleaning Tasks’ and ‘Monitoring Tasks’ sections. Use the dashboard to manage outstanding tasks and participant visit data.

Use the filters at the top of the page to adjust the task list by site, participant, or other criteria.

Viewing Data Cleaning Tasks​

The ‘Data Cleaning Tasks’ section on the EDC dashboard shows outstanding tasks for the selected site and role, grouped by participants, CRFs, and statuses.

CRFs pill​

The CRFs pill shows forms that require review and are in ‘To Review’ or ‘Cleaning’ status for the selected site.

  • Use filters at the top of the dashboard to narrow the CRFs list.
  • Select a CRF to open it in a modal for review.

Pending Exceptions pill​

The PENDING EXCEPTIONS pill shows forms, where a form exception has been requested but not yet granted or denied, for the selected site. Once a decision is made, the CRF no longer appears under PENDING EXPECTATIONS.

  • Use filters at the top of the dashboard to narrow the pending exceptions list.
  • Select a CRF to open it in a modal for review.

Queries pill​

The QUERIES pill shows ‘On Hold’ or ‘Answered’ queries for the selected site.

  • Use filters at the top of the dashboard to adjust the queries list.
  • Select a query to open it in a modal for quick review and closure.
  • Select a CRF to open it in a modal for review.

Viewing monitoring tasks​

The ‘Monitoring Tasks’ section on the EDC dashboard shows outstanding tasks for the selected site and role, grouped by participants, CRFs, and statuses.

CRFs pill​

The CRFs pill shows forms that require verification and are in ‘To Review’ or ‘Cleaning’ status for the selected site.
